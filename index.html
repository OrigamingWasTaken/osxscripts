OSXScripts

Welcome to this little repo of Mac scripts you can easily use in the terminal!

How to use

To use these scripts, you will need curl and sh . Here is an exemple:

$ curl s.origaming.sh/finder | sh -s 10 5

This will create 10 finder windows and delete them after 5 seconds. But you can change it with arguments.

Scripts List

Here is a list of scripts and their argument. Always use them like: $ curl s.origaming.ch/{script}

Script Arguments
finder w: number of windows, d: time to wait before clearing the windows (-1 to never clear)

You can also do: "$ curl s.origaming.ch/help | sh" to get a pretty manual in the terminal
